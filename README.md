# Sudoku
A hard coded low level sudoku game written in c++ using [ncurses](https://en.wikipedia.org/wiki/Ncurses)

#### Compile
```
  $ g++ sudoku.cpp -o sudoku -lncurses
```

#### Run
```
  $ ./sudoku
```
